### Задача 1
---
Напишите функцию, которая переворачивает строку.</br>
[Решение, код на C++](./Solutions/task01.cpp)

```C++
std::string reverseString(std::string str)
{
    std::string temp = "";
    for (int i = str.length(); i >= 0; i--)
    {
        temp += str[i];
    }
    return temp;
}
```
